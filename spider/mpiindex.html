MPI_Cart_coords函数

  • 项目
  • 2022/09/22

本文内容

  1. 语法
  2. 参数
  3. 返回值
  4. Fortran
  5. 要求
  6. 另请参阅

确定在组中给定排名的笛卡尔拓扑中的进程共同点。

语法

c++ 
int MPIAPI MPI_Cart_coords(
   MPI_Comm               comm,
   int                    rank,
   int                    maxdims,
   _Out_cap_(maxdims) int *coords
);

参数

  • 通讯
    具有笛卡尔结构的 Communicator。

  • rank
    在 一组命令内的进程排名。

  • maxdims
    调用程序中矢量 共同点 的长度。

  • coords
    包含指定进程的笛卡尔坐标的整数数组 (大小最大值) 。

返回值

返回 成功MPI_SUCCESS 。 否则，返回值是错误代码。

在 Fortran 中，返回值存储在 IERROR 参数中。

Fortran

FORTRAN 
    MPI_CART_COORDS(COMM, RANK, MAXDIMS, COORDS, IERROR)
        INTEGER COMM, RANK, MAXDIMS, COORDS(*), IERROR

要求

产品

HPC Pack 2012 MS-MPI 可再发行组件包、HPC Pack 2008 R2 MS-MPI 可再发行组件包、HPC Pack 2008 MS-MPI 可再发行组件包或 HPC Pack 2008 客户端实用工具

标头

 Mpi.h;Mpif.h

库

 Msmpi.lib

DLL

 Msmpi.dll
